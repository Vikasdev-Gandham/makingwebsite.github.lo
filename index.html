HTML Color

html colors are specifed with predefined color mames, RGB,HEX,HSL,RGBA,or HSLA Values.

RGB:Red,Green,Blue these colors refer to system for representing the color to be used on a computer display.Red Green And Blue can be combined in various proportion to obtain any color. True color is an RGB color model standard that specifies 256 shades for red, green and blue spaces, totaling 16 million colors, much more than what the human eye can distinguish, which is only 10 million colors

HEX:A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.
